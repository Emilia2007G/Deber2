# Deber2
## Nombre Completo:
 Carla Guaman
## NRC
2684
## Descripcion
### Este programa solicita el nombre del estudiante y dos notas, calcula el promedio de esas notas y muestra en pantalla el nombre y el promedio obtenido

##logica
El programa inicia.

Solicita al usuario que ingrese su nombre.

Pide que ingrese la primera nota.

Pide que ingrese la segunda nota.

Calcula el promedio sumando las dos notas y dividiendo el resultado para 2.

Muestra en pantalla el nombre del estudiante y el promedio calculado.

El programa finaliza.
